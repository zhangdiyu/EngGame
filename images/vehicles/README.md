## 推荐图片来源

1. **Flaticon** (https://www.flaticon.com)
   - 搜索 "car icon", "truck icon" 等
   - 选择 PNG 格式，建议尺寸 128x128 或 256x256
   - 免费下载需注明出处

2. **Icons8** (https://icons8.com)
   - 有大量免费交通工具图标
   - 支持 PNG 下载

3. **Freepik** (https://www.freepik.com)
   - 搜索 "vehicle cartoon" 获取卡通风格图片

4. **OpenClipart** (https://openclipart.org)
   - 完全免费的剪贴画资源

## 图片要求

- 格式：PNG（透明背景最佳）
- 尺寸：建议 128x128 像素或更大
- 风格：卡通/简笔画风格，适合幼儿
- 文件名：必须与上方列表完全一致（小写，用连字符）

## 自动占位

如果图片不存在，程序会自动显示一个灰色占位符图标。
